# wmf
wine's minimal fetch

![](_res/showcase.png)

## configuration
simple configuration can be done in `src/config.h`, for anything more "complex"
look at `src/main.c`.

after editing the config recompile.

## installation
`git clone https://github.com/wine/wmf && cd wmf && make && make install`
